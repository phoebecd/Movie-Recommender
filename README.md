# MovieMatch ◈

MovieMatch is a contextual movie recommendation platform designed to help you find the perfect film based on your current mood, company, and energy—not just your watch history.

## Features
- **Contextual Discovery**: Get recommendations tailored to who you're with and how you feel.
- **Taste Profiles**: A lightweight TF-IDF based ML engine that learns your film preferences.
- **Friend Blending**: Combine your taste with friends for the perfect movie night.
- **Full-Stack Suite**: Built with React, Firebase, and FastAPI.

## Getting Started
See [SETUP.md](./SETUP.md) for detailed instructions on how to install and run the project locally.

## Tech Stack
- **Frontend**: React 18, TypeScript, Tailwind CSS, Framer Motion.
- **Backend / Auth / DB**: Firebase suite (Hosting, Functions, Firestore, Auth).
- **Inference Engine**: Python FastAPI microservice using Scikit-Learn.
- **Metadata Enrichement**: TMDB API.
